Project Information

  • Category: CAD and Optimization
  • Class: AAE 251: Introduction to Aerospace Design
  • Project Dates: 8/21/2023 - 12/9/2023

Description

The final project for my Introduction to Aerospace Design class had me design/prototype an Unmanned Aerial Vehicle (UAV) from scratch for use in search and rescue applications. Throughout the course I learned many aspects of aircraft design, including atmospheric considerations, aircraft sizing, mission diagrams, and fuel calculations. I also learned how to analyze these using MATLAB and how to optimize them for the various mission constraints provided. This resulted in moderately sized, fixed wing UAV taking inspiration from the Hermes 450. The prototype was modeled using Siemens NX and sized using various MATLAB scripts tuned to mission-specific parameters.
