Description

As part of my summer internship at the GTRI ATAS lab, I got to program Arduino Braccio robot arms with 6 degrees of freedom to complete various tasks autonomously. Due to COVID restrictions, I would program this robot with the goal of deploying the code on real robots in the lab remotely. We used Vision Based Sensing, primarily based on OpenCV and April Tags to detect and locate objects in 3D space. We also developed our own kinematics and inverse kinematics libraries tuned to Braccio robot parameters for motion planning. Finally, ROS was used to integrate the vision and kinematics to motion profile and execute actions autonomously. This all culminated in a final presentation where the robot picked up a box with an April Tag on it autonomously and placed it in a target location.
